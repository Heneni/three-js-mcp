# Anna Mills Clone wshit

Animated, scroll-reactive collage site deployable on Vercel.

## To update images:
- Replace `public/artwork.csv` with your new image URLs.
- Push changes to your repo.
- Vercel auto-builds, and your site updates.

## Local development
```
bash
npm install
npm run dev
```
